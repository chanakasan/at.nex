# @ CLI - Script Runner

### Install
**one liner**
```
curl -fsSL https://raw.githubusercontent.com/chanakasan/at.nex/main/setup/web-21.sh | bash
```

**manual**
```
mkdir -p ~/dotfiles
git clone https://github.com/chanakasan/at.nex ~/dotfiles/at.nex
sh ~/dotfiles/at.nex/setup/install.sh 
```

### Usage

***built-in***
```
# smoke test
@ hello

# edit bashrc
@ ed

# reload bashrc
@ rel
```
